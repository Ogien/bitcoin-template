# bitcoin-template
Simple skeleton for a SPA

# Install
## Full Install
Running in Terminal, `npm run full-install`, will install the following
- **Gems**
  - Sass
  - Bourbon
- **Bower Dependencies**
- **NPM Dependencies**
- **Bourbon Files**

## Install Gems
Run in Terminal, `npm run install-gems`

## Install Packages
Run in Terminal, `npm run install-packages`

#Run the Application
Run in Terminal, `npm start`
